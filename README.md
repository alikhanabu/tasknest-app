# TaskNest Demo Project

Учебный демо-проект для практики React.

## Описание
Проект отображает список досок (boards) в виде карточек.
Данные хранятся в массиве объектов и отображаются динамически.

## Используемые технологии
- React
- Vite
- Tailwind CSS
- SCSS

## Запуск проекта
1. Клонировать репозиторий
2. Установить зависимости:
   npm install
3. Запустить проект:
   npm run dev
